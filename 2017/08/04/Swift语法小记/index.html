Swift语法小记

发表于 2017-08-04 | 分类于 Swift | 阅读次数

最近我在学习Swift，通过The Swift Programming language对Swift这门语言有了一个大致的了解，因此准备写一篇文章对Swift的语法以及相关的语言特性做一个小结，顺便复习一下。

一、标准类型

1. 常见类型

Swift内置的常见标准类型为:

  • 整数(Int..)
  • 浮点数(Float、Double…)
  • String
  • Bool

2. 可选类型

可选类型是Swift独有的类型，其有两层含义:

  • 一是变量是否有值(没有值时变量为nil)
  • 二是变量的值是多少

我们可以在变量类型后加一个问号或者感叹号来表示这是一个可选类型的变量，如下所示:

使用问号和感叹号的区别是：

  • 问号声明的可选类型变量，在使用时需要使用感叹号对其进行显式解包
  • 感叹号声明的可选类型变量，在使用时Swift会对其自动进行隐式解包，因此不需要加感叹号

需要注意的是不论是显示解包还是隐式解包，只要对值为nil的变量进行解包就会触发运行时错误！因此，在使用可选类型的变量之前，最好对其进行nil检查。常见的检查方法有:

  • 直接判断变量是否为nil
  • 使用可选绑定(optional binding)
